# Project Brief

## Overview
This project is a GameLab platform for uploading, managing, and publishing games.

## Core Requirements
1. Users can register, authenticate, and manage profiles.
2. Game uploads via multer to server/uploads/games/.
3. Serve uploaded games statically.
4. Moderation system for game approval.
5. Multi-language support (i18n).

## Scope
- Backend: Node.js + Express, SQLite database.
- Frontend: vanilla HTML/CSS/JS, no frameworks.
- Authentication: JWT and bcrypt.
- Moderation roles: user, moderator, admin.

## Success Metrics
- Successful game upload and display.
- Moderator actions (approve/reject) functional.
- Responsive UI across devices.

## Future Enhancements
- Analytics dashboard.
- Community forums.
- Game rating system.
